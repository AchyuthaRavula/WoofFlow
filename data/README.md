This project does NOT store real customer data.

WoofFlow processes customer emails from PlayStaytion Pet Resort in real time
via Gmail integration. For privacy and security reasons, no raw email data is
stored in this repository.

---

## Data Types Used (Conceptual)

- Incoming customer emails (text)
- Extracted entities:
  - Customer name
  - Pet type
  - Dates & services
- Classified intent:
  - Boarding
  - Daycare
  - Grooming
  - Training
  - Cancellation
  - General inquiry

---

## Privacy & Security
- No real customer emails are included
- No personally identifiable information (PII) is stored
- All examples in this repository are synthetic and anonymized

---

## Where to See Examples
Sanitized example inputs and outputs are available in the `examples/` folder.
